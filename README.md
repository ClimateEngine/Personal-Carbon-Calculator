# 📖 Project Summary

Climate change is one of the greatest global challenges of our time, and individual lifestyle choices contribute significantly to greenhouse gas emissions.

The **Personal Carbon Footprint Calculator** is an interactive web application developed using **Python, Streamlit, and Plotly** to help users estimate their annual carbon emissions based on transportation, household energy consumption, dietary habits, air travel, and lifestyle choices.

The application not only calculates emissions but also provides clear visualizations and actionable sustainability recommendations, encouraging users to make environmentally responsible decisions.
